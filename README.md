## 📚 技术架构

### 🎵 VITS 2.0 声音模型
```mermaid
graph LR
A[语音输入] --> B(声纹特征提取)
B --> C{多尺度分析}
C --> D[韵律建模]
C --> E[音素建模]
D --> F[波形生成]
E --> F
F --> G[高保真输出]
```

- **多语言支持**：中文/英语/日语/方言混合生成
- **噪声抑制**：支持60dB环境噪声过滤
- **情感控制**：内置8种情感模式（喜悦/愤怒/悲伤等）
